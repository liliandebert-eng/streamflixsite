
# StreamFlix Premium UI

Interface e-commerce moderne inspirée de l'univers Netflix réalisée avec :

- React
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Installation

```bash
npm install
npm install framer-motion lucide-react
npm run dev
```

## Stack recommandée

- Vite + React
- Tailwind CSS

## Fonctionnalités

- Catalogue dynamique
- Fiche produit animée
- Design Netflix premium
- Responsive mobile / desktop
- Glow rouge et animations fluides
- Carrousel d'offres
- Top ventes
